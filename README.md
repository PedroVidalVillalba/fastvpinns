# FastVPINNs

A robust tensor-based deep learning framework for solving PDE's using Variational Physics-Informed Neural Networks (VPINNs). 

## Installation

To install the package, run the following command:

```bash
pip install fastvpinns
```

## Usage

```python
import fastvpinns
```
